About

This web-page and backend application are both running on a single Raspberry Pi. The web-page was built using spring-boot, and the backend was built using spring & Java.

A huge thanks to the Pi4J library for making Java on the Raspberry Pi a possibility as well as spring for your ease of use, and integration. You have made one happy turtle.
